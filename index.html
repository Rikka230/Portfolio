Abstract Harmony

Project Details

  • Medium: Acrylic on canvas
  • Dimensions: 36" x 48"
  • Year: 2023
  • Client: Private Collector

Description

This abstract composition explores the harmony between geometric precision and organic fluidity. Created for a private collector's residence, the piece uses a carefully balanced color palette to create visual tension while maintaining overall cohesion. The work took approximately 80 hours to complete, with multiple layers of paint creating depth and texture.

Mythical Guardian

Project Details

  • Software: Blender, Substance Painter
  • Polycount: 85,000 tris
  • Year: 2023
  • Client: Game Development Studio

Description

The Mythical Guardian was created as a boss character for an upcoming fantasy RPG. The design blends elements of Eastern and Western mythological creatures, with intricate armor detailing and realistic fur simulation. The model is fully rigged and optimized for real-time rendering in game engines. Textures were painted in Substance Painter with 4K resolution maps.

Lumina Branding

Project Details

  • Scope: Full Brand Identity
  • Deliverables: Logo, Color Palette, Typography, Stationery
  • Year: 2022
  • Client: Lumina Brands

Description

The Lumina rebranding project involved creating a sophisticated visual identity for this luxury skincare company. The logo combines a custom wordmark with an abstract sun motif, representing the brand's focus on natural illumination and radiance. The color palette uses soft golds and warm neutrals to convey premium quality. The complete identity system includes business cards, packaging, and marketing materials.

Verde E-commerce

Project Details

  • Platform: Custom Shopify Theme
  • Features: Product Filtering, AJAX Cart, Custom Checkout
  • Year: 2023
  • Client: Verde Organic

Description

The Verde e-commerce platform was designed to showcase organic home goods with an emphasis on sustainability and craftsmanship. The clean, minimalist interface puts products front and center, with intuitive navigation and seamless checkout flow. Custom UI elements include an interactive product configurator and lifestyle imagery integration. The design system ensures brand consistency across all digital touchpoints.

Portrait Series

Project Details

  • Medium: Digital Painting
  • Software: Photoshop, Procreate
  • Year: 2023
  • Client: Personal Project

Description

This portrait series explores the interplay of light and texture across diverse faces. Each piece was created using custom digital brushes to achieve painterly effects while maintaining photographic realism. The collection was featured in Digital Arts Magazine's "Emerging Artists" showcase. The works demonstrate advanced understanding of facial anatomy, color theory, and emotional expression through portraiture.

Modern Villa

Project Details

  • Software: 3ds Max, V-Ray, Photoshop
  • Render Time: 12 hours per image
  • Year: 2023
  • Client: Architectural Firm

Description

This luxury villa visualization was created for a high-end residential development in California. The photorealistic renders showcase the property's clean lines, expansive glass walls, and seamless indoor-outdoor living spaces. Careful attention was paid to material textures, lighting conditions, and landscaping details to accurately represent the architect's vision. The images were used for pre-construction marketing and helped secure several off-plan purchases.
